เอาใหม่ฉันว่าเราต้องมาออกแบบใหม่
อย่างแรกฉันต้องการเว็ปไซต์ที่เหมือนกับ Google Colab แต่ให้สร้างชื่อฟังก์ชั่นทุกอย่างเป็นภาษาอังกฤษนะแต่ฉันจะอธิบายเป็นชื่อภาษาไทย
ส่วนบนสุดจะประกอบไปด้วย ไฟล์ แก้ไข มุมมอง แทรก รันไทม์ และความช่วยเหลือ
ซึ่งฉันจะบอกคุณว่าแต่ละหมวดหมู่ฟังก์ชั่นมีฟังก์ชั่นอะไรบ้าง

ไฟล์ ในหมวดหมู่นี้จะเป็นฟังก์ชั่นสำหรับจัดการแก้ไขหรือบันทึกไฟล์ Notebook Jupiter ที่แก้ไขอยู่หรือที่เคยใช้งาน จากในรูปคุณจะเห็นว่า
1.1ฟังก์ชั่นแรกเป็นค้นหาในไดรฟ์สร้างฟังก์ชั่นนี้ไว้แค่ UI ก่อนยังไม่ต้องทำอะไรเพราะเรายังไม่ได้ทำระบบ login และเชื่อม google drive
1.2ต่อจากค้นหาในไดรฟ์จะเป็นเปิดในโหมด Playground เป็นระบบสำหรับทดสอบสิ่งต่างๆและโค้ดที่เขียนในโหมดนี้จะไม่ถูกบันทึกอัตโนมัติสร้างไว้แค่ UI เหมือนกับ ค้นหาในไดรฟ์เพราะเรายังไม่ได้สร้างระบบบันทึกอัตโนมัติ
1.3 สมุดบันทึกใหม่ในไดรฟ์ เป็นการสร้าง Notebook Jupiter ใหม่ในไดรฟ์ของ user สร้างไว้แค่ UI เหมือนกันเพราะยังไม่ได้ทำระบบเชื่อม google drive
1.4 เปลี่ยนชื่อ จากในรูปภาพเราจะเห็นว่าไฟล์ Notebook Jupiter นี้มีชื่อว่า Untitled3.ipynb ซึ่งเมื่อเรากดฟังก์ชั่นเปลี่ยนชื่อจะทำให้เราสามารถแก้ไขชื่อไฟล์ Notebook Jupiterได้ และอีกอย่างหนึ่งเมื่อ user กด Double Click ที่ชื่อไฟล์ทำให้ผู้ใช้เปลี่ยนชื่อได้เหมือนกันด้วย
1.5 ย้าย เป็นฟังก์ชั่นจัดการที่อยู่ไฟล์ในไดรฟ์สร้างไว้แค่ UI พอ
1.6ย้ายลงถังขยะ เป็นฟังก์ชั่นจัดการที่อยู่ไฟล์ในไดรฟ์ให้ย้ายไปที่ถังขยะสร้างไว้แค่ UI พอ
1.7 บันทึกสำเนาในไดรฟ์ บันทึกไฟล์ Notebook Jupiter นี้ไว้ในไดรฟ์สร้างไว้แค่ UI พอ
1.8 บันทึกสำเนาเป็น GitHub Gist สร้างไว้แค่ UI พอยังไม่ได้ทำระบบเชื่อมต่อกับ github
1.9 บันทึกสำเนาใน GitHub สร้างไว้แค่ UI พอยังไม่ได้ทำระบบเชื่อมต่อกับ github
1.10 บันทึก บันทึกไฟล์ Notebook Jupiter ให้ code เป็น version ล่าสุด และเมื่อผู้ใช้กด Ctrl+S จะเป็นการสั่งการฟังก์ชั่นนี้อัตโนมัติ
1.11บันทึกและปักหมุดเวอร์ชันที่มีการแก้ไขนี้: บันทึกเวอร์ชันปัจจุบันของโน้ตบุ๊กและปักหมุดไว้ เพื่อให้สามารถย้อนกลับมาดูได้ในภายหลัง และหากผู้ใช้กด Ctrl + M S จะเป็นการสั่งใช้งานฟังก์ชั่นนี้
1.12ประวัติการแก้ไข: ดูประวัติการแก้ไขของโน้ตบุ๊ก
1.13ดาวน์โหลด: ดาวน์โหลดโน้ตบุ๊กในรูปแบบต่างๆ เช่น .ipynb, .py, .html
